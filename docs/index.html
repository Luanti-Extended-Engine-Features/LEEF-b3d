A library containing hopefully all you need to read, solve and navigate a b3d. Aswell as (eventually) write one.

Modules

b3d_reader parse .b3d files into a lua table.
b3d_writer writes b3d models in the same format as outputted by the b3d reader module
b3d_nodes allows you to get information about nodes (bones or meshes) within a b3d table (generated with b3d_reader) located in mtul.b3d_nodes.
